Computational Geometry Robust Geometric Primitives Input description: A point $p$ and line segment $l$, or two line segments $l_1$, $l_2$. Problem description: Does $p$ lie over, under, or on $l$? Graph Problems : Hard Problems Dealing with NP-completeness. Clique Input description: A graph $G = (V,E)$. Problem description: What is the largest $S \subset V$ such that for all $x, y \in S,, (x, y) \in E$? Finding the maximum clique is NP-complete, as hard as it gets.
